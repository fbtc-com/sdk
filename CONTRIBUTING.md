# Contributing

Thank you for your interest in contributing to the Function FBTC Agent SDK!

## Setup

```bash
yarn install
yarn build
```

## Development

- Use yarn (never npm)
- Run `yarn format` and `yarn lint` before committing
- Conventional commits (`feat:`, `fix:`, `chore:`)

## Testing

```bash
yarn test:required
```

## Pull Requests

Keep PRs focused. Include a short summary and test plan.
